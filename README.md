# CHIP-8 Emulator

This is a simple CHIP-8 Emulator for the CASIO FX-CP400.

## Installation

1. Install hollyhock-2 from [SnailMath](https://github.com/SnailMath/hollyhock-2)
2. Copy the .bin to /fls0/chip_8_emulator.bin
3. Copy the ROMs to /fls0/chip8/roms/rom.ch8

You can download ROMs from the internet or this repository: [chip8-roms](https://github.com/kripod/chip8-roms).

## Usage

- Load the .bin file via the hollyhock-2 launcher
- Choose a ROM from the DropDown menu
- Press load
- To exit the emulation press [CLEAR]
- If there was an error the screen will be most likely black and you need to press a key to return
