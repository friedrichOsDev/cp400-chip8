# CHIP-8 Emulator

This is a simple CHIP-8 Emulator for the CASIO FX-CP400.

## Installation

1. Install hollyhock-2 from [SnailMath](https://github.com/SnailMath/hollyhock-2)
2. Copy the .bin to `/fls0/chip_8_emulator.bin`
3. Place ROM files in `/fls0/chip8/roms/`. You can download ROMs from [chip8-roms](https://github.com/kripod/chip8-roms).

## Usage

- Launch `chip_8_emulator.bin` via hollyhock-2.
- Select a ROM from the dropdown and press **Load**.
- **[CLEAR]**: Exit emulation.
- If an error occurs (screen goes black), press any key to return.
